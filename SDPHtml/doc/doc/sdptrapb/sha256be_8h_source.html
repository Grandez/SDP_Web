48 /*
49  * Description:
50  * This file implements the Secure Hash Algorithms
51  * as defined in the U.S. National Institute of Standards
52  * and Technology Federal Information Processing Standards
53  * Publication (FIPS PUB) 180-3 published in October 2008
54  * and formerly defined in its predecessors, FIPS PUB 180-1
55  * and FIP PUB 180-2.
56  *
57  * A combined document showing all algorithms is available at
58  * http://csrc.nist.gov/publications/fips/
59  * fips180-3/fips180-3_final.pdf
60  *
61  * The five hashes are defined in these sizes:
62  * SHA-1 20 byte / 160 bit
63  * SHA-224 28 byte / 224 bit
64  * SHA-256 32 byte / 256 bit
65  * SHA-384 48 byte / 384 bit
66  * SHA-512 64 byte / 512 bit
67  *
68  * Compilation Note:
69  * These files may be compiled with two options:
70  * USE_32BIT_ONLY - use 32-bit arithmetic only, for systems
71  * without 64-bit integers
72  *
73  * USE_MODIFIED_MACROS - use alternate form of the SHA_Ch()
74  * and SHA_Maj() macros that are equivalent
75  * and potentially faster on many systems
76  *
77  */
